Privacy Policy

Vua Tho Media Hub is a local desktop/web application for internal media review and content planning.

TikTok data used

With explicit user authorization, the app uses TikTok Login Kit and Display API to read basic profile information and public video metadata for the authorized user, including cover image URLs.

Storage

OAuth tokens and imported media metadata are stored locally on the user's machine. This GitHub Pages website does not store access tokens, refresh tokens, client secrets, or media data.

Use

Imported TikTok metadata and cover images are used only for local review, deduplication, scoring, and internal content planning. The app does not automatically post to TikTok.
